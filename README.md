# 🩺 MedicAI

MedicAI is a simple AI-based symptom checker web app built with Flask and Ollama.

---

## 🚀 Features
- Symptom analysis using local AI models
- Fast response system (Phi / Mistral)
- Basic emergency detection
- Simple web interface

---

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- Ollama (Local AI models)

---

## 📂 Project Structure
- app.py → Flask backend
- templates/index.html → Frontend UI
- .gitignore
